Step 1: Vectors and Vector Spaces

A vector is an element of a vector space—a mathematical structure where you can add vectors and multiply them by scalars. In quantum mechanics, quantum states are vectors in a complex vector space (Hilbert space). A vector can be written as |v⟩ = (v₁, v₂, ..., vₙ) or in Dirac notation as |v⟩ = Σ vᵢ|i⟩. Vectors have magnitude (length) and direction. Two fundamental operations are vector addition |v⟩ + |w⟩ and scalar multiplication c|v⟩ where c is a complex number.

Vector: |v⟩ = (v₁, v₂, ..., vₙ) where vᵢ ∈ ℂ
Addition: |v⟩ + |w⟩ = (v₁+w₁, v₂+w₂, ..., vₙ+wₙ)
Scalar multiplication: c|v⟩ = (cv₁, cv₂, ..., cvₙ)

Step 2: Inner Products and Norms

The inner product ⟨v|w⟩ = Σ vᵢ*wᵢ (where * denotes complex conjugate) measures the "overlap" between two vectors. It's a complex number that generalizes the dot product. The norm ||v|| = √⟨v|v⟩ = √(Σ |vᵢ|²) is the length of the vector. A unit vector has ||v|| = 1. Two vectors are orthogonal if ⟨v|w⟩ = 0. In quantum mechanics, orthogonal states represent distinguishable states, and the inner product ⟨φ|ψ⟩ gives the probability amplitude for transitioning from |ψ⟩ to |φ⟩.

Inner product: ⟨v|w⟩ = Σ vᵢ*wᵢ ∈ ℂ
Norm: ||v|| = √⟨v|v⟩ = √(Σ |vᵢ|²) ∈ ℝ
Orthogonality: ⟨v|w⟩ = 0 ⟺ v ⊥ w

Step 3: Matrices and Linear Transformations

A matrix A is a rectangular array of numbers that represents a linear transformation—a function that maps vectors to vectors while preserving addition and scalar multiplication. Matrix-vector multiplication Av transforms vector v into a new vector. In quantum mechanics, matrices represent operators (observables and gates). An n×n matrix has n² entries. Matrix multiplication (AB)v = A(Bv) represents composition of transformations. The identity matrix I satisfies Iv = v for all v.

Matrix: A = [aᵢⱼ] where aᵢⱼ ∈ ℂ
Matrix-vector product: (Av)ᵢ = Σⱼ aᵢⱼvⱼ
Linear transformation: A(c₁v₁ + c₂v₂) = c₁Av₁ + c₂Av₂

Step 4: Basis and Dimensionality

A basis is a set of linearly independent vectors that span the entire vector space—every vector can be written as a unique linear combination of basis vectors. For a 2D qubit, the computational basis is {|0⟩, |1⟩}, so any state is |ψ⟩ = α|0⟩ + β|1⟩. The dimension of a space is the number of basis vectors needed. An n-qubit system lives in a 2ⁿ-dimensional complex vector space. Changing basis is like changing perspective—the same quantum state looks different in different bases (e.g., computational vs. Hadamard basis).

Basis: {|e₁⟩, |e₂⟩, ..., |eₙ⟩} linearly independent and spanning
Any vector: |v⟩ = Σ cᵢ|eᵢ⟩ (unique coefficients cᵢ)
Dimension: dim(V) = n (number of basis vectors)
