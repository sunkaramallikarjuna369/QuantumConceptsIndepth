Step 1: Hermitian Matrices (Observables)

A matrix Â is Hermitian if = Â, where † denotes the conjugate transpose (also called adjoint). For a 2×2 matrix, this means A₁₁ and A₂₂ are real, and A₁₂ = A₂₁*. Hermitian matrices represent observables (measurable quantities) in quantum mechanics—position, momentum, energy, spin, etc. The key property: all eigenvalues of a Hermitian matrix are real, which is essential since measurement outcomes must be real numbers. The Pauli matrices σₓ, σᵧ, σᵤ are Hermitian and represent spin measurements along x, y, z axes. The spectral theorem guarantees that Hermitian matrices have orthonormal eigenvectors, forming a complete basis for measurement.

Step 2: Unitary Matrices (Quantum Gates)

A matrix Û is unitary if Û†Û = I (identity matrix), equivalently Û† = Û⁻¹. Unitary matrices represent reversible quantum operations—quantum gates and time evolution. The key property: unitary matrices preserve the norm ||Û|ψ⟩|| = ||ψ⟩||, ensuring that normalized states remain normalized (probabilities sum to 1). All quantum gates (Hadamard, CNOT, Toffoli) are unitary. Time evolution under the Schrödinger equation is unitary: Û(t) = e^(-iĤt/ℏ). Unitary matrices have eigenvalues on the unit circle: |λᵢ| = 1, often written as λᵢ = e^(iθᵢ). This reversibility is fundamental to quantum computation.

Unitary: Û†Û = I (inverse equals conjugate transpose)
Norm preservation: ||Û|ψ⟩|| = ||ψ⟩||
Hadamard: H = (1/√2)[[1, 1], [1, -1]] with H† = H = H⁻¹

Step 3: Relationship Between Hermitian and Unitary

Every Hermitian matrix can generate a unitary matrix via exponentiation: Û = e^(iÂ). This is how observables (Hermitian) generate time evolution (unitary). For example, the Hamiltonian Ĥ (Hermitian, represents energy) generates time evolution Û(t) = e^(-iĤt/ℏ). Conversely, every unitary Û can be written as Û = e^(iĤ) for some Hermitian Ĥ. A matrix that is both Hermitian and unitary must satisfy = Â and Â² = I, meaning its eigenvalues are ±1. The Pauli matrices σₓ, σᵧ, σᵤ are both Hermitian (observables) and unitary (self-inverse gates). This duality is central to quantum mechanics.

Step 4: Computing Properties

To check if a matrix is Hermitian: compute † (conjugate transpose) and verify = Â. To check if unitary: compute Û†Û and verify it equals I. For eigenvalues: solve det(Â - λI) = 0. For a 2×2 matrix [[a, b], [c, d]], the conjugate transpose is [[a*, c*], [b*, d*]]. The determinant of a unitary matrix has |det(Û)| = 1 (often det(Û) = 1 for special unitary matrices like quantum gates). The trace of a Hermitian matrix is real: Tr(Â) ∈ ℝ. These computational checks verify whether a matrix can represent a valid quantum gate (unitary) or observable (Hermitian).
