Quantum States and State Vectors

Explore Dirac notation, bra-ket representation, and quantum state vectors

Input

Process

Step 1: Dirac Notation

Quantum states are represented using Dirac notation (bra-ket notation):

|ψ⟩ = α|0⟩ + β|1⟩ (ket vector)
⟨ψ| = α*⟨0| + β*⟨1| (bra vector)

Where |ψ⟩ is a ket (column vector) and ⟨ψ| is a bra (row vector, complex conjugate transpose).

Step 2: State Vector Representation

In the computational basis, a quantum state can be written as a column vector:

|ψ⟩ = [α]
      [β]

This is a vector in a 2-dimensional complex Hilbert space.

Step 3: Normalization

For a valid quantum state, the norm must equal 1:

⟨ψ|ψ⟩ = |α|² + |β|² = 1

This ensures that probabilities sum to 1.

Step 4: Basis Transformations

The same quantum state can be expressed in different bases. For example, the Hadamard basis relates to the computational basis by:

|+⟩ = (|0⟩ + |1⟩)/√2
|−⟩ = (|0⟩ − |1⟩)/√2
