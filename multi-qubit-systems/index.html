Step 1: Tensor Product and Hilbert Space

Multi-qubit systems are described by the tensor product of individual qubit Hilbert spaces. For n qubits, the state space is H = H₁ ⊗ H₂ ⊗ ... ⊗ Hₙ, which has dimension 2ⁿ. A two-qubit system has basis states |00⟩, |01⟩, |10⟩, |11⟩, where |ij⟩ = |i⟩ ⊗ |j⟩. A general two-qubit state is |ψ⟩ = α|00⟩ + β|01⟩ + γ|10⟩ + δ|11⟩, where |α|² + |β|² + |γ|² + |δ|² = 1. This exponential growth (2ⁿ dimensions for n qubits) is the source of quantum computational power—a classical n-bit register has 2ⁿ possible states, but only one is realized at a time, while a quantum n-qubit register can be in a superposition of all 2ⁿ basis states simultaneously. However, measurement collapses the state to one outcome, so extracting information requires clever algorithms. The tensor product structure means operators act on individual qubits: (A ⊗ B)|ψ⟩⊗|φ⟩ = A|ψ⟩ ⊗ B|φ⟩.

Hilbert space: H = H₁ ⊗ H₂ ⊗ ... ⊗ Hₙ, dim(H) = 2ⁿ
Two-qubit state: |ψ⟩ = Σᵢⱼ cᵢⱼ|ij⟩, Σ|cᵢⱼ|² = 1
Tensor product: |i⟩ ⊗ |j⟩ = |ij⟩

Step 2: Entanglement and Separability

A multi-qubit state is separable (product state) if it can be written as |ψ⟩ = |ψ₁⟩ ⊗ |ψ₂⟩ ⊗ ... ⊗ |ψₙ⟩. For example, |+⟩⊗|0⟩ = (|0⟩+|1⟩)/√2 ⊗ |0⟩ = (|00⟩+|10⟩)/√2 is separable. If a state cannot be written as a product, it is entangled. The Bell states are maximally entangled two-qubit states: |Φ⁺⟩ = (|00⟩+|11⟩)/√2, |Φ⁻⟩ = (|00⟩-|11⟩)/√2, |Ψ⁺⟩ = (|01⟩+|10⟩)/√2, |Ψ⁻⟩ = (|01⟩-|10⟩)/√2. Measuring one qubit of a Bell state instantly determines the other, regardless of distance—this is the EPR paradox and Bell's inequality violation. For three qubits, the GHZ state |GHZ⟩ = (|000⟩+|111⟩)/√2 and W state |W⟩ = (|001⟩+|010⟩+|100⟩)/√3 are inequivalent types of entanglement. Entanglement is a resource for quantum computing, quantum cryptography, and quantum teleportation. Quantifying entanglement uses measures like concurrence, entanglement entropy, and negativity.

Separable: |ψ⟩ = |ψ₁⟩ ⊗ |ψ₂⟩ (product state)
Bell state: |Φ⁺⟩ = (|00⟩+|11⟩)/√2 (maximally entangled)
GHZ state: |GHZ⟩ = (|000⟩+|111⟩)/√2
W state: |W⟩ = (|001⟩+|010⟩+|100⟩)/√3

Step 3: Multi-Qubit Gates and Circuits

Multi-qubit gates create and manipulate entanglement. The CNOT (controlled-NOT) gate is the fundamental two-qubit gate: CNOT|control,target⟩ = |control, target ⊕ control⟩, where ⊕ is XOR. CNOT flips the target qubit if the control is |1⟩. Starting from |00⟩, applying H⊗I (Hadamard on first qubit) gives (|00⟩+|10⟩)/√2, then CNOT produces (|00⟩+|11⟩)/√2 = |Φ⁺⟩, a Bell state. The Toffoli gate (CCNOT) is a three-qubit gate that flips the target if both controls are |1⟩—it's universal for classical computation. The SWAP gate exchanges two qubits: SWAP|ij⟩ = |ji⟩. Any multi-qubit unitary can be decomposed into single-qubit gates and CNOTs (universal gate set). Quantum circuits are sequences of gates acting on qubits, read left-to-right. Parallelism arises from superposition: a quantum circuit on n qubits processes 2ⁿ inputs simultaneously, but measurement extracts only one output. Clever algorithms (Shor's, Grover's) exploit interference to amplify correct answers.

CNOT: |c,t⟩ → |c, t⊕c⟩
Bell state preparation: (H⊗I)CNOT|00⟩ = |Φ⁺⟩
Toffoli: |c₁,c₂,t⟩ → |c₁,c₂, t⊕(c₁∧c₂)⟩
Universal: Any U = (single-qubit gates) + CNOT

Step 4: Applications in Quantum Computing and Information

Multi-qubit systems enable quantum algorithms that outperform classical ones. Shor's algorithm factors integers in polynomial time using quantum Fourier transform on n qubits, threatening RSA cryptography. Grover's algorithm searches unsorted databases in O(√N) time (quadratic speedup). Quantum simulation uses multi-qubit systems to simulate other quantum systems (chemistry, materials science)—Feynman's original motivation for quantum computing. Quantum error correction encodes logical qubits into multiple physical qubits to protect against decoherence (e.g., Shor's 9-qubit code, surface codes). Quantum cryptography (QKD) uses entangled photon pairs for secure key distribution. Quantum teleportation transfers quantum states using entanglement and classical communication. Quantum metrology uses entangled states (GHZ, NOON states) for precision measurements beyond the standard quantum limit. Current quantum computers (IBM, Google, IonQ) have 50-1000 qubits, but decoherence and gate errors limit circuit depth. Achieving fault-tolerant quantum computing requires millions of physical qubits for error correction.

Shor's algorithm: Factor N in O((log N)³) time
Grover's algorithm: Search N items in O(√N) queries
Error correction: Logical qubit = f(physical qubits)
Quantum supremacy: 53 qubits (Google, 2019) for specific task
